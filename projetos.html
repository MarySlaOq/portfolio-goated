Sabores do Sol

Uma pastelaria fictícia que usa plantas comestíveis como o seu principal meio de decoração de bolos.

Foi criada para o Projeto de Aptidão Profissional para o meu ano final no Colégio de Gaia.

Para este projeto foram criados diversos conteúdos e informações como um logotipo totalmente original e um website independente que funciona através da linguagem de programação PHP.

The character arquive

Este projecto consiste num aplicação web JavaScript com uma pequena base de dados.

É uma ferramenta de escrita direcionada a armazenar e relacionar personagens de historias criadas pelo utilizador

Além de personagens, outros aspectos caracterizadores do mundo podem também ser definidos, como a geografia do mundo, um calendario de eventos do mundo, e uma linha temporal de eventos.

Rumble

A animação Rumble foi uma animação feita em grupo entre mim e mais dois colegas da licenciatura de multimédia.

Com Edgar Carneiro e Tomás Sarreira, tivemos o intuito de participar da edição de 2024 da Mad Games Jam, então tem um tema relacionado com jogos de vídeo sobre aventura.
